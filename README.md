# Open FAIR Samples

Website for the **Open FAIR Samples** project: *Open and FAIR samples: maturing the sample data ecosystem*.

Live site: https://linked-samples.github.io/open-fair-samples/

## About

This project is funded by the U.S. National Science Foundation under the Findable Accessible Interoperable Reusable Open Science (FAIROS) program. The project develops four community roadmaps — with researchers, data repositories, sample repositories, and scholarly publishers — to advance the adoption of persistent identifiers and leading metadata practices for scientific samples.

**PI:** Natalie H. Raia | **Co-PI:** Andrea K. Thomer | University of Arizona

## Getting Started

This site was forked from [Geo-FAIR/geofair](https://github.com/Geo-FAIR/geofair).

### Setup for GitHub Pages

1. Fork or clone this repository
2. Go to **Settings → Pages**
3. Set source to **Deploy from a branch** → `main` → `/ (root)`
4. Your site will be live at `https://YOUR-ORG.github.io/open-fair-samples/`

### Editing

- Edit any page directly in GitHub's web editor
- Site updates within ~1–2 minutes after commit
- No build step needed — just static HTML/CSS

### Local Preview

Open any HTML file directly in a browser — no server required.
