# Open and FAIR Samples

Website for NSF FAIROS Project: *Open and FAIR samples: maturing the sample data ecosystem*.

Live site: https://linked-samples.github.io/fair-sample-ecosystem/

## About

This project is funded by the U.S. National Science Foundation under the Findable Accessible Interoperable Reusable Open Science (FAIROS) program. The project develops four community roadmaps — with researchers, data repositories, sample repositories, and scholarly publishers — to advance the adoption of persistent identifiers and leading metadata practices for scientific samples.

**PI:** Natalie H. Raia | **Co-PI:** Andrea K. Thomer | University of Arizona
